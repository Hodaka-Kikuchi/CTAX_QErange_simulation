# CTAX Q-E Range Simulator

Interactive simulator for the accessible Q-E range of the CTAX triple-axis spectrometer at ORNL.

## Features

- Powder mode
- Single-crystal mode
- Interactive reciprocal lattice visualization
- Adjustable lattice constants and scattering plane
- Plotly-based interactive figures
